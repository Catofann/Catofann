# 使用Github pages搭建个人博客



## 1.注册一个Github账号

打开[Github](https://github.com/)，注册一个账号.

![注册界面](https://pic.imgdb.cn/item/61454ea62ab3f51d91fc1cc1.jpg "Github注册界面")

# 2.创建一个新的仓库

找到界面右上角`+`按键，创建一个新仓库.

![新建仓库](https://pic.imgdb.cn/item/61458f952ab3f51d915b2a17.jpg,"新建仓库")

如下图所示设置仓库.

![选项](https://pic.imgdb.cn/item/6145913a2ab3f51d915d875a.jpg,"仓库设置")

下载[Github桌面版](https://desktop.github.com/)，选择克隆仓库，将刚刚创建的仓库克隆到本地.

![克隆仓库](https://pic.imgdb.cn/item/614592ce2ab3f51d915fc75e.jpg)

打开[模板网站](http://jekyllthemes.org/)选择一个心仪的博客模板下载，若不熟悉jekyll，注意选择安装方式为克隆即可使用的模板.

找到仓库本地路径（一般为C:\Users\计算机名\Documents\GitHub\仓库名），将下载的模板文件复制到文件夹中.

打开Github桌面版，选择提交`Commit`更改，然后进行同步`Fetch origin`。

同步结束后，打开[Github](https://github.com/)，进入仓库，选择右上角设置`Setting`.

![设置](https://pic.imgdb.cn/item/614595ac2ab3f51d9163e4f9.jpg)

如下图所示开启Github Pages功能.

![设置](https://pic.imgdb.cn/item/614597fe2ab3f51d916736c5.jpg)

等待几分钟即可打开自己的博客了！博客网址默认为：用户名.github.io.
